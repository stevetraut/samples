Custom Insert Wizard Sample Overview

This sample is an example of a fully-custom dialog for inserting a Java control. The control itself isn't functional. The interesting code is in the implementation of ControlWizard and the dialog that accompanies it.

As you use the "Name the Control" tab of the wizard dialog, the dialog's bottom pane will display messages that describe how your choices impact what the wizard does.
