Running the Chapter 11 POTranslator Web Service

The POTranslator web service uses JavaScript to accept an XML message of one shape and translate it to another.

Use the following step to test the web service.

  1. Start WebLogic Workshop and WebLogic Server.
  2. Open POTranslator.jws and click the start button.
  3. When Test View appears, return to WebLogic Workshop and open POXML.txt.
  4. Copy the contents of POXML.txt and paste them into the XMLMap box under submitOrder.
  5. Click the submitOrder button.
  6. After Test View refreshes, compare the contents of POXML.txt with the Service Request and Service Response you see in Test View.
